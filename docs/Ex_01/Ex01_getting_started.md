# Exercise 01: Explore the Fabric data engineering experience

In this exercise, you'll act as Miguel the data engineer. You'll transfer Zava's data from ADLS Gen2 into a lakehouse and begin to prepare the data for the upcoming merger between Zava and Litware Inc.

This exercise includes the following tasks:  

- Create a Microsoft Fabric-enabled workspace and add a lakehouse
- Create a Shortcut to reference external data
- Create Delta tables by using a Spark notebook

Estimated time to complete this exercise: **25 minutes**
