# Exercise 0: Provision and configure resources

## Scenario
Zava is preparing its analytics environment to integrate external data from Litware. Before any analytics or AI workloads can run, foundational Azure resources and secure access must be configured. In this exercise, you'll set up identity, access, and storage components required for the rest of the lab.

## Objectives
In this exercise, you'll prepare the lab environment by performing the following tasks:  
- Configure an account service principal and secret
- Add a client secret
- Add a role assignment to an Azure Storage account
- Upload data to a storage account

## Duration
Estimated time to complete this exercise: **10 minutes**
