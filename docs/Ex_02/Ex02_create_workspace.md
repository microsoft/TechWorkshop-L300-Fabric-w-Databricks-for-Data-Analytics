# Exercise 02: Integrate Azure Databricks with Fabric 
Zava acquired Litware Inc. The company uses Azure Databricks and stores their churn and sales data in ADLS Gen2. 

This exercise shows how Zava uses Microsoft Fabric with Databricks to integrate their data with data from Litware. You'll see how Unity Catalog helped Zava's data architects quickly understand Litware Inc.'s data.
