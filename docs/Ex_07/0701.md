## Task 01: Create an Azure Databricks AI/BI Genie assistant


### Introduction
Azure Databricks Genie provides conversational analytics directly over curated datasets.

### Description
In this task, you'll create a Genie assistant and connect it to curated data.

### Example scenario
Business users ask natural language questions to explore customer churn trends.

### Success criteria
The Genie assistant returns valid responses and SQL queries.

### Learning resources
-   Databricks Genie overview
-   Conversational analytics patterns

### Key steps

#### 01: Create the assistant

1. Open Edge and go to [Microsoft Azure](portal.azure.com).

1. If prompted, sign in.

1. Search for and then select `Azure Databricks`.

    ![96ufaar4.jpg](../../media/96ufaar4.jpg)

1. Select the Azure Databricks resource that was provisioned for you.

    ![y6jzbtkb.jpg](../../media/y6jzbtkb.jpg).

1. Select **Launch workspace**.

1. In the left pane, in the **SQL** section, select **Genie**.

    ![1maru3ow.jpg](../../media/1maru3ow.jpg)

1. Paste the numbers into a notepad file for later use.

1. On the command bar, select **+ New**.

    ![98agcbe7.jpg](../../media/98agcbe7.jpg)

1. In the **Connect your data** dialog, select **dbkwks@lab.LabInstance.Id**, select **schema@lab.LabInstance.Id_processeddata** and then select **silver_customerchurn_data**.

    ![bl9u7bsr.jpg](../../media/bl9u7bsr.jpg)

1. Select **Create**.

    ![uzkteuk8.jpg](../../media/uzkteuk8.jpg)

1. At the top of the page, in the **New space** field, enter `Litware_Genie`.

    ![om0cursg.jpg](../../media/om0cursg.jpg)


1. Submit the following prompt:

    `What is the distribution of customers by Gender?`

1. Review the response.

    ![jxsv7t4s.jpg](../../media/jxsv7t4s.jpg)

1. Select **Show code**.

    ![0a4rpw18.jpg](../../media/0a4rpw18.jpg)

1. Review the SQL query that was used to generate the results.

    ![wk6gajtv.jpg](../../media/wk6gajtv.jpg)

    1. At the top of the page, in the address bar, copy the GUID that appears after **genie/rooms/** up to **/chats**. This is the Genie Space ID.

### 02: Enable the Managed MCP servers preview
In this task, you'll add your Genie MCP server as a tool within Microsoft Foundry.

1. On the Azure Databricks page, select your username avatar and then select **Previews**.

    ![15euzoa2.jpg](../../media/15euzoa2.jpg)

1. On the **Previews** pane, in the **Filter previews** field, enter `MCP`.

	{: .warning }
    > In newer deployments, the feature may be enabled by default.

    ![5lk7xyen.jpg](../../media/5lk7xyen.jpg)

1. Set **Managed MCP Servers** to **On**.

    ![zaemx17i.jpg](../../media/zaemx17i.jpg)
