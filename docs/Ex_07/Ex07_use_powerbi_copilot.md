# Exercise 07: Create AI-driven insights by using Microsoft Foundry and Genie

## Scenario
Zava wants to combine governed analytics data with conversational AI to deliver insights through enterprise-grade agents.

## Objectives
In this exercise, you'll: 
- Configure an Azure Databricks AI/BI Genie assistant.
- Connect the assistant to Microsoft Foundry. 
- Expose the insights through a Foundry agent.

## Duration
Estimated time to complete this exercise: **30 minutes**.
